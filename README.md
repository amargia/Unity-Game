# Unity-Game

# I love Chocolate
It's a game where you have to retrieve the golden chocolate bar from the evil guards who took it.

# Controls
* Arrows (or A, W, S, D) and Q, E - Move
* K - Shoot
* I - Inventory
* C - Change cameras
* R - Raycast shooter
* To activate the elevators you need to step on the buttons next to them
